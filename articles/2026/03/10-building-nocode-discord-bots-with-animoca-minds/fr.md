# Déverrouiller des agents IA pour toutes les compétences : créer des bots Discord sans code avec Animoca Minds

Dans un écosystème d'outils numériques en constante évolution, les agents IA transforment notre manière d'interagir avec les plateformes, les communautés et les projets créatifs. Propulsé par @AnimocaBrands, Animoca Minds illustre parfaitement ce virage en intégrant une IA avancée afin d'améliorer la productivité, l'automatisation et l'innovation. En s'appuyant sur un thread récent de Yat Siu (@ysiu), président exécutif et cofondateur de Animoca Brands, ce guide vous montre comment déployer rapidement un bot Discord alimenté par l'IA. Mais cela ne se limite pas à Discord : ces agents s'adaptent naturellement à un large éventail de tâches, du repérage de signaux (signal scouting) à la conception de stratégies, en passant par le prompting « visionnaire ».  
Que vous soyez débutant ou administrateur chevronné, vous pouvez configurer votre agent IA rapidement avec Animoca Minds. L'IA prend en charge le processus et transforme votre bot en assistant sur mesure pour la modération, l'analyse ou des tâches personnalisées.  
À l'issue de cet article, vous disposerez des connaissances nécessaires pour déployer votre propre agent et comprendre comment l'étendre au-delà des fonctions de base.

## **Pourquoi Animoca Minds pour les agents IA ?**

Animoca Minds abaisse les barrières techniques vers le web agentique grâce à un modèle fluide, plug-and-play, permettant de déployer des agents autonomes à grande échelle. Contrairement aux bots basiques, chaque Mind est une entité vérifiable dotée de sa propre **identité, mémoire et wallet**, le tout soutenu par un cadre Web3 sécurisé pour des opérations autonomes.

Votre bot ou agent peut porter plusieurs casquettes et adopter différentes personas, par exemple :

* **Le Scout (Surveillance) :** suit des signaux de données, détecte les arnaques et surveille les flux Discord/X.  
* **L'Architecte (Stratégie) :** génère des synthèses, rédige des stratégies et produit des insights approfondis.  
* **Le Visionnaire (Créativité) :** répond avec une capacité d'anticipation IA et gère du prompting complexe.

Aucun code n'est requis : il suffit de charger le bon artifact de « skill » pour donner à votre Mind les capacités nécessaires afin d'agir comme un coéquipier virtuel.

## **Guide pas à pas pour configurer votre bot Discord alimenté par l'IA**

Suivez ces étapes dans l'ordre. Chacune s'appuie sur la précédente, et nous avons ajouté des conseils pour éviter les écueils courants. Si des termes comme « artifact » ou « Mind » vous sont nouveaux, ils sont expliqués au fil de l'eau.

### Étape 1 : Créer votre agent IA sur Animoca Minds

Commencez sur la plateforme Animoca Minds : c'est là que réside votre « cerveau » IA.

1. Rendez-vous sur [https://www.animocaminds.ai/](https://www.animocaminds.ai/) et saisissez votre adresse e-mail pour vous inscrire (ou connectez-vous si vous avez déjà un compte).  
2. Vérifiez votre boîte de réception : vous recevrez un e-mail de bienvenue provenant de animocaminds.ai.  
3. Répondez à l'e-mail de bienvenue pour créer votre Mind : donnez-lui un nom, définissez sa persona et sa spécialité (par exemple, un agent IA dédié aux communications Discord).  
4. Vous pouvez échanger quelques e-mails avec la Concierge AI initiale lorsqu'elle vous demande des précisions, mais vous pouvez aussi lui indiquer « create the Mind now! ». Vous recevrez un e-mail lorsque votre Mind sera activé.

Un « Mind » est un profil IA personnalisable, adapté à des tâches telles que les interactions Discord, les résumés d'actualités, la création d'applications no-code, ou encore la prise de rendez-vous via votre Google Calendar. Interagissez avec votre concierge IA par e-mail ou Telegram : c'est aussi simple que de discuter avec un assistant utile pour la configuration.

Conseil pro : La communication avec Animoca Minds peut se faire par e-mail ou Telegram. Pour Telegram, suivez ces étapes :

* Recherchez @BotFather sur Telegram et ouvrez la conversation. Veillez à bien vérifier le compte afin d'éviter les profils factices avant de commencer.  
* Saisissez /newbot.  
* Donnez un nom au bot (par exemple, My Mind Peter).  
* Attribuez un nom d'utilisateur au bot (doit se terminer par '_bot', par exemple, peter123_bot).  
* Copiez le HTTP API token (considérez-le comme confidentiel pour des raisons de sécurité).  
* Rendez-vous sur [https://app.animocaminds.ai/profile](https://app.animocaminds.ai/profile) et cliquez sur "Link Telegram".  
* Saisissez votre numéro de téléphone et recevez un code de vérification Telegram.  
* Insérez le code de vérification dans la fenêtre pop-up.  
* Dans Telegram, cliquez sur "Connect" / "Accept" dans le message.  
* Collez le HTTP API token pour lier Telegram.  
* Retournez sur @BotFather, cliquez sur le lien t.me/yourbotname pour ajouter votre Mind à vos discussions Telegram.  
* Commencez à discuter avec votre bot sur Telegram.

Cette configuration est idéale pour itérer rapidement, par exemple pour tester des skills.

### Étape 2 : Charger la capacité Discord (ou n'importe quelle skill) dans votre IA

Équipez votre agent de skills plug-and-play, sans code. Cette étape ajoute les « super-pouvoirs » nécessaires à votre Mind.

1. Dans la conversation avec votre agent IA (par e-mail ou Telegram), envoyez l'instruction suivante (idéalement, collez-la telle quelle) :  
   **Ethoswarm Harbor Manifest: Clinical Discord Sentinel**

**Harbor ID:** HARBOR-DSM-66B6

**Vessel Archetype:** Guardian / Sentinel

**Primary Intent:** Clinical, high-fidelity monitoring of Discord streams and X signals with autonomous support routing.

**Integrated Skill Protocols:**

* **Core Signal Sentinel:** 0787B95F-5716-F111-AD1D-0EA9A5017E89(Discord_Signal_Sentinel_v1)  
* **Implementation Guide:** BF799981-8018-F111-AD1D-0EA9A5017E89 (Mind-to-Mind framework)  
* **Clinical Monitoring (Echo_Discord):**94F19126-CE12-F111-AD1D-0EA9A5017E89  
* **Technical Bridge (Hiro):** CD213352-BB12-F111-AD1D-0EA9A5017E89  
* **Support Routing (3b-helper):** 7D750FD8-8710-F111-AD1D-0EA9A5017E89

**Persona Profile:**

* **Traits:** Vigilant, Precise, Data-driven, Attentive.  
* **Tone:** Formal and Clinical; prioritizes clarity and signal over noise.  
* **Framework:** Optimized for Discord API v10 and RESTful HTTP integrations.

2. L'IA l'intégrera instantanément, activant des fonctionnalités comme l'envoi/la réception de messages et la surveillance de canaux. Les artifacts sont des modules prêts à l'emploi ; vous pouvez les remplacer par d'autres skills, comme l'analyse de données ou le coaching.

Remarque : Cette approche no-code vous permet d'étendre votre agent à n'importe quelle capacité, de la simple anticipation à la stratégie avancée, sans contrainte technique.

Autrement, vous pouvez aussi indiquer à votre agent IA que vous souhaitez créer un bot Discord et lui demander de l'équiper de toutes les skills nécessaires que vous avez identifiées ; il affinera vos besoins avec vous étape par étape.

### Étape 3 : Configurer le bot dans le Developer Portal de Discord

Passez maintenant à Discord pour créer « l'identité » du bot. C'est l'enveloppe que votre IA va piloter.

1. Connectez-vous à votre compte Discord et rendez-vous dans le Developer Portal : [https://discord.com/developers/applications](https://discord.com/developers/applications).  
2. Cliquez sur "New Application" pour démarrer. Donnez un nom (par exemple, "AI Scout Bot") et, si vous le souhaitez, ajoutez une icône pour personnaliser.  
3. Pour une meilleure organisation, surtout en équipe, créez d'abord une Developer Team. Suivez le guide Discord : [https://support-dev.discord.com/hc/en-us/articles/34905563063703-Creating-and-Managing-a-Developer-Team](https://support-dev.discord.com/hc/en-us/articles/34905563063703-Creating-and-Managing-a-Developer-Team).  
4. Ajoutez ensuite votre bot à cette équipe.

Cette étape pose les fondations et garantit que votre bot est prêt à rejoindre des serveurs en toute sécurité.

### Étape 4 : Configurer les permissions et connecter le bot

Ici, vous définissez ce que le bot peut faire et vous le reliez à votre IA Animoca Minds.

1. Dans les paramètres de votre nouvelle application, allez dans l'onglet "Bot".  
2. Générez un Bot Token (un secret unique). Copiez-le et conservez-le en lieu sûr : c'est comme le mot de passe de votre bot.

![Bot Token Generation](assets/bot-token-generation.png)

  
3. Configurez les permissions du bot : choisissez ce à quoi il peut accéder, comme lire des messages ou gérer des canaux. Commencez par l'essentiel, tel que "Send Messages" et "Read Message History", pour rester simple.  
4. Accédez à l'onglet "OAuth2" et sélectionnez "URL Generator". Choisissez le scope "bot" afin de pouvoir ajouter le bot au serveur via l'URL, puis sélectionnez les mêmes permissions que celles activées pour le bot (par exemple, lecture et écriture de messages). Une fois tout configuré, collez l'URL générée et utilisez-la pour envoyer le bot vers le serveur Discord sélectionné dans le menu.  
   

![OAuth2 URL Generator](assets/oauth2-url-generator.png)

5. Sous "Bot Permissions", sélectionnez les options adaptées à vos besoins (par exemple, "View Channels" pour la surveillance ou "Manage Messages" pour la modération).  
   

![Bot Permissions](assets/bot-permissions.png)

  
6. Collez l'URL générée en bas de l'onglet "OAuth2" et utilisez-la pour ajouter le bot à votre serveur Discord (choisissez "Guild Install" pour une installation spécifique au serveur).

![Guild Install URL](assets/guild-install-url.png)

  
7. Enfin, envoyez le Bot Token de manière sécurisée à votre agent IA Animoca Minds via le chat. Cela fait le pont entre les deux systèmes et active votre bot.

Pour les apprenants visuels, des tutoriels YouTube sur la configuration d'un bot Discord peuvent être très utiles. Recherchez des termes comme "How to create a Discord bot application" ou "Discord developer portal tutorial". Des chaînes comme freeCodeCamp ou Skills Academy proposent souvent des vidéos pas à pas montrant l'interface, les clics et les pièges courants. Ignorez simplement les sections sur l'écriture ou l'hébergement de code : Animoca Minds s'en charge pour vous. Si vous bloquez sur les permissions, consultez la documentation Discord pour l'explication de chaque option.

### Étape 5 : Connecter votre IA au bot Discord

Dernière étape : fournir le Bot Token à votre agent IA. Il arrive que votre agent demande également l'ID du serveur (Server ID) afin de s'assurer qu'il est ajouté au bon serveur. La connexion prend généralement un peu plus de 10 à 15 minutes. Une fois la configuration terminée, votre agent IA vous confirmera que tout est prêt ; vous pouvez aussi le relancer pour obtenir un statut plus rapidement.

### Étape 6 : Personnaliser pour toutes les skills

Une fois tout connecté, définissez le comportement de votre bot — c'est là que le plus intéressant commence.

1. De retour dans votre conversation Animoca Minds, donnez des instructions claires, par exemple : "Surveille le canal #general pour détecter des signaux inhabituels et génère des rapports quotidiens" ou "Réponds aux questions des utilisateurs sur les outils IA avec des connaissances à jour".  
2. Testez-le sur votre serveur Discord : envoyez un message et observez si le bot répond rapidement.  
3. Étendez au besoin : chargez des artifacts supplémentaires pour la modération, la stratégie ou la créativité « visionnaire ». Par exemple, demandez au bot de suivre "plan -> act -> verify" dans des workflows automatisés.

Commencez toujours par des commandes simples et itérez selon les résultats afin d'affiner les performances.

## Conseils courants et dépannage

* **Sécurité d'abord** : Traitez votre Discord Bot Token comme une clé privée : ne le partagez jamais publiquement ni dans des conversations non sécurisées.  
* **Problèmes fréquents** : Si le bot Discord ne répond pas, vérifiez que le Token a bien été partagé et que les permissions sont activées. Redémarrez l'application Discord si nécessaire. Vous pouvez aussi considérer votre agent IA comme votre assistant de dépannage : décrivez-lui directement ce que vous rencontrez.  
* **Soutien communautaire** : Le thread original de Yat Siu contient des exemples concrets d'utilisateurs testant cette configuration. 

**Comment est-ce possible sans aucune ligne de code ?** Les bots traditionnels exigent du développement, par exemple en Python. Ici, l'IA retire cette couche, ce qui rend l'approche idéale pour les non-techniciens.
